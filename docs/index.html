Plan

A scheme from your own shelf

Drop two photos into the chat: the bare mini and the reference you are chasing. The agent reads both, works the paint scheme zone by zone, and grounds every pick in the pots you already own. Where there is a real gap it names one or two worth buying, never a fresh set. Your collection grows on purpose.

Without it: a correct but generic answer built on a random paint set.

Restock

Log paints without typing

Snap your paint rack or paste a screenshot of your last order. minipainter recognizes the products and adds them to your inventory. No names to type, no catalog to build by hand. Your shelf stays current with almost no effort.

One photo in, your inventory back out.

Quickstart

# clone & enter
$ git clone https://github.com/arturskowronski/warpaint-cli
$ cd warpaint-cli

# search the catalog
$ node src/cli.mjs paint search bone
$ node src/cli.mjs paint search black --json

# track what's on your rack
$ node src/cli.mjs inventory own "Abaddon Black"
$ node src/cli.mjs inventory list

# match by color or description
$ node src/cli.mjs match color "#d2c29b"
$ node src/cli.mjs match describe bone

# open the workshop ledger TUI
$ node src/cli.mjs tui

The real terminal UI: node src/cli.mjs tui
