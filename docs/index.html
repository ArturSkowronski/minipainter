Get going

Quickstart

# clone & enter
$ git clone https://github.com/arturskowronski/warpaint-cli
$ cd warpaint-cli

# search the catalog
$ node src/cli.mjs paint search bone
$ node src/cli.mjs paint search black --json

# track what's on your rack
$ node src/cli.mjs inventory own "Abaddon Black"
$ node src/cli.mjs inventory list

# match by color or description
$ node src/cli.mjs match color "#d2c29b"
$ node src/cli.mjs match describe bone

# open the workshop ledger TUI
$ node src/cli.mjs tui
